# Tic-Tac-Toe 🎮

A simple 2-player multi-turn tic-tac-toe game with a visual interface.

## ✨ Features

- 3x3 game board  
- Turn-based logic  
- Win and tie detection  
- UI reset button  
- Fully responsive design for mobile devices

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
